# Security Policy

## Supported versions

The most recent `main` branch is supported.

## Reporting a vulnerability

Please report vulnerabilities privately before public disclosure.

1. Do not open a public issue for security vulnerabilities.
2. Contact the maintainer directly through private repository channels.
3. Include:

- Affected area
- Reproduction steps
- Potential impact
- Suggested mitigation (if known)

You will receive acknowledgment as soon as possible, followed by validation and remediation updates.

## Data handling notes

Planner Buckets stores data in browser local storage by default.
Review exported JSON files carefully before sharing.
Clipboard copy actions place selected task text on your system clipboard only after you click a copy control.
